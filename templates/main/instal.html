Installation Guide

For the time being, ore is in early alpha release and does not provide installation binaries, this means you will have to compile the language from source in order to use it.

installing dependencies

in order to compile ore, you will have to make sure a few dependencies are installed first. sudo apt install build-essential git make cmake gcc ccache

Getting the source

Next, you need to get the ore source code from the github page using git. git clone --recurse-submodules https://github.com/itskarudo/ore.git && cd ore

compiling the library and REPL

Let's create a directory for our build binaries and compile! mkdir Build && cd Build cmake .. && make

installation

Now that the compilation is complete, we can install the ore language by running sudo make install
