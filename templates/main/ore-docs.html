Ore Programming Language Guide (v0.1.0)

Your syntax files need to be encoded in standard ASCII

Comments

Single-line comment start with --. Multiline comments are not yet supported so they must be divided into mutliple Single-line comments

-- this is a comment 


-- this is 
-- a 
-- multiline comment

Variables

there is not a keyword for variable declaration , Variables are declared by the name and the value If you try to access a variable that wasn’t initialized before, a runtime exception is thrown.

variable_1 = "test"
variable_2 = true

Literals

Ore has a total of 9 primitive types. 

25             : Number
25.5           : Number
"Charly"       : String
[1, 2, 3]      : ArrayObject
nil            : Nil
false          : Boolean
true           : Boolean
function() {}  : FunctionObject
{prop : true } : Object

You can get the type of a value by calling the type() function. It returns the name of the type as a string.

type(25) -- => "Number"
type("Hello World!") -- => "String"
type(nil) -- => "Nil"
type(func() {}) -- => "FunctionObject"

nil

The Nil type can be compared to undefined in javascript or nil in some other languages.

nil

Boolean

A Boolean only has two different values: true and false.and they are case sensitive, for example “True” is not supported. 

true
false

Numbers

All Numeric types inside Ore are C++’s native double type.

Numeric literals, just like any other expression in the language, can be prefixed with - to negate them.They can be prefixed by “0b”,"0x”,"0o” to specify the radix. 

binary = 0b11 -- 2
octal  = 0o10 -- 8 
hex    = 0x10 -- 16
dec    = 10

Floats are created using the . character.

123.45689

A Numeric will throw an exception if you pass the lower or upper limit of double. The following REPL session elaborates this:

Operations such as 1 / 0 will result in an exception.

String

A String represents an immutable sequence of ASCII characters.

It uses C++’s native std::string type underneath.

You can create it using either " or 'characters.

"hello world"
'hello world!'

special characters inside the string are not supported.

"\"" -- => \"

A string can’t span multiple lines.

Array

An Array is a resizeable list of items of any type. It is typically created with an array literal:

[1, 2, 3]
[1, "hello world", ["whats up"]]

You can add new items to an array using the append method:

nums = []
nums:append(0)
nums:append(1)
nums:append(2)

nums -- => [0, 1, 2]

Objects

Ore objects are declared similarly to JavaScript objects: 

Test = {
  a : 1,
  b : function () {
          return(5)
      }
  }

Test.a -- => 1
Test.b() -- => 5

You can access properties of objects via []:

Box = {
  name : "mybox"
}

Box["name"] -- => "mybox"

Functions

You can define a new function like this:

function foo() {
  return("hello world")
}

or like this 

test = function () {
        print("Test")
    }

You can also do explicit returns

func foo() {
  if true {
    return(true)
  }

  return(false)
}

foo() -- => true

If you need the function literal, you can use anonymous function literals:

function foo(callback) {
  callback(42)
}

foo(
    function (args) {
            print(args)
        } 
)

anonymous functions can be called directly. 

function () {print(5)}()

Assignments

Assignment is done with the = character or by composing it with another binary expression character. 

var = 1 -- 1 
var += 1  -- 2 
var -= 5 -- -3 
var *= -1 -- 3

all the characters for Assignment are : =,+=,-=,*=,/=,<<=,>>=,..=

Control expressions

All control expressions inside Ore are statements.

This means that you can’t place a control expression inside a method call.

If statement

cond = true
if cond {
        print("valid!")
} else {
       print("invalid")
}

While statement

counter = 0
while counter < 5 {
    print(counter)
    counter += 1
}

For statement

for(i=5;i<25;i+=1){
        print(i)
    }

in the for,if and while statement if one statement is called, the curly braces can be omitted.

Do-While statement

i = 0 
do {
        print(i)
        i += 1
    } while i < 25

the braces in the do while loop are obligatory.

Truthy and falsey values

A truthy value is a value that is considered true for an if and while guard. A falsey value is a value that is considered false in those places.

The only falsey values are false ,nil and 0. Any other value is truthy.

Types and methods

The next sections will assume you know what object oriented programming is, as well as the concepts of classes and methods.

Everything is an object

Everything in Ore is an object. Not every type can have an internal state however. Only Objectand Array can have an internal state.

When you write 5, the interpreter actually treats it as a primitive. There are no funny castings or object instantiations (inside Ore). When you write 5.times, the interpreter searches for a primitive class called NumberObject and checks if it contains a method called times.

This allows the interpreter to reuse the same object for all primitives of the same type.

This principle applies to all language primitives. The primitive class Array for example, specifies a method called append which inserts an element into the array.

[] -- array index and object property access

Exceptions

You can throw exceptions from anywhere in the program. Everything is throwable.

function foo(arg) {
if arg < 10 {
  throw("ValueException","arg is smaller than 10")
}
}

foo(5)

This will show up in the console as:

Backtrace (most recent calls first):
  foo
  global
------------------------------
ValueException: arg is smaller than 10
nil

and exceptions can be handled as follows : 

try{
 -- code
} catch(e) {
 -- code
 -- the exception e has two attributes  : e.type and e.message
} finally {
 -- code
}

Requiring files

You can include other files using the import method. It accepts a single string argument that serves as the filename.

When including a file, the contents of the export variable is then returned by the require call.

Example: 

math = import("libmath.so")
math.cos(5)

Command line arguments and flags

The orepl command has the ability to receive flags. A list of these can be obtained by running orepl -h in the command line.

You can pass flags via the following format

orepl -d -p -g -a input.ore

Arguments after the input file name are passed to the program instead of the interpreter.

orepl input.ore arg1 arg2 arg3

You can access command line arguments and flags via the args array.

Built-in REPL

Ore has a built-in REPL (Read Eval Print Loop) where you can quickly test out new ideas.

You can invoke it by typing orepl

Here you can write Ore statements. Run them by pressing enter.

IO

Things related to IO and the file system are directly included in the prelude. 

print("hello world")
var = input()
